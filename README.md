# saucedemo-qa-assessment
A QA assessment for Enyata

Project Overview

This repository contains:
- Exploratory testing documentation
- UI automation tests
- API tests using Postman

Tools Used
- Cypress / Selenium
- Postman
- Node.js
- GitHub


Features Tested
### Exploratory Testing
- Login
- Add to cart
- Remove from cart
- Product filtering
- View cart
- Checkout
### UI Automation
Automated high-priority user flows for the Saucedemo application.
### API Testing
Postman tests for the Users endpoint.



##How To Run UI Tests

1. Clone the repository
```bash
git clone (https://github.com/maedu-spec/saucedemo-qa-assessment.git)
```

2. Install dependencies
```bash
npm install
```

3. Run tests
```bash
npx cypress open
```

OR
```bash
npx cypress run
```

## How To Run API Tests

1. Open Postman
2. Import the Postman collection from the `postman` folder
3. Run the collection
